# ベータ版リリース 0.9.0.0
* 2019-12-15  
  
 + セットアップ手順  
---------------------------------------------------------------
  1.ProcessManagement.zipをダウンロードし、解凍してください。  
  2.インストールは不要です。ProcessManagement.exeを実行してください。
  
 + 使用方法  
---------------------------------------------------------------  
  1. 起動したらプロジェクトを必要な項目を入力し、新規作成してください。  
     2. プロジェクト名  
     2. 担当者名  
     2. 期間（日付～日付）  
      
  1. 工程の追加
     2. 挿入したい欄で右クリック  
     2. 個別工程 or 共通工程を選択  
      
  1. 工程期間の変更・移動  
     2. マウスドラッグで移動＆サイズ変更できます。
    
  1. 工程内容の編集
     2. 工程をダブルクリック  
     2. タイトル・メモ・色を編集  
     2. 工程別にフォルダを自動作成するので、必要なファイルはここに保存できます。  
      
  1. 工程の削除  
     2. 右クリックで削除できます。
